# Проект. Telegram-бот. Задание 2. Блок-схема с разметкой

Задание выполняется на основе блок-схемы, которую вы составили в задании 1. Если вы получили замечания к предыдущему заданию, обязательно учтите их в своей блок-схеме прежде чем приступать к выполнению этого задания.

## Задание

1. Дополните свою блок-схему. Теперь на ней должны быть отмечены 3 категории случаев:
• бот отправляет сообщение пользователю;
• бот получает ответ пользователя с кнопки (как в конспекте 7);
• бот получает текстовое сообщение с клавиатуры.

    Отмечать эти варианты удобнее всего будет цветами, не забудьте сделать легенду: подписать, какой цвет что означает.

2. Каждый из отмеченных случаев будет реализован отдельной функцией (каждая отправка сообщения и каждое получение ответа от пользователя: разные ответы на один вопрос - это одна функция). Придумайте для каждой из этих функций уникальное название, которое будет отражать ее суть.
